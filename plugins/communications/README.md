# communications

Claude skills for the public information office. Built for PIOs, communications directors, social media coordinators, and the staff who run the city's public voice.

## Skills in this plugin

| Skill | Purpose |
|---|---|
| [press-release-drafter](skills/press-release-drafter) | Drafts a press release in standard inverted-pyramid form with quote placeholder |
| [citizen-newsletter-builder](skills/citizen-newsletter-builder) | Drafts a citizen newsletter with section structure, calls to action, and city events |
| [social-media-municipal-style](skills/social-media-municipal-style) | Drafts social media posts in a neutral municipal voice that won't get the city in trouble |
| [talking-points-writer](skills/talking-points-writer) | Drafts council member or department head talking points on a contested topic |
| [crisis-comms-template](skills/crisis-comms-template) | Drafts initial holding statements and follow-up updates for a public crisis |

## Install

```
/plugin install communications@whitegloveai-gov
```

## Compliance and tone

Citizen-facing output produced by these skills includes the TRAIGA-required AI-interaction disclosure where applicable. The municipal-style social media skill is deliberately conservative — public-records-safe, politically neutral, and unlikely to invite unintended liability.

The crisis-comms template is a starting point. Real crisis communication requires a human in the loop, coordination with legal and public safety, and chain-of-command sign-off.

## Notes on Examples

Examples in this plugin reference the fictional City of Cedar Ridge, TX for continuity across skills. The skill logic is jurisdiction-agnostic — adapt example specifics to your city.
