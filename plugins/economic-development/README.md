# economic-development

Claude skills for economic development. Built for ED corporation staff, business retention and expansion (BR&E) coordinators, site selection liaisons, and the small-shop ED director who wears every hat.

## Skills in this plugin

| Skill | Purpose |
|---|---|
| [incentive-proposal-drafter](skills/incentive-proposal-drafter) | Drafts a Chapter 380 / 381 incentive agreement proposal narrative with project, performance metrics, and clawback structure |
| [site-selection-response](skills/site-selection-response) | Drafts a response to a confidential site selection RFI with workforce, sites, utilities, and incentive sections |
| [business-retention-survey-analyzer](skills/business-retention-survey-analyzer) | Analyzes BR&E survey results and identifies risk and opportunity signals |
| [annual-report-narrative](skills/annual-report-narrative) | Drafts the narrative sections of an ED corporation annual report |

## Install


## Compliance

Texas economic development incentives are governed by Local Government Code Chapters 380, 381, and 504/505 (Type A and Type B sales tax corporations), as well as Tax Code Chapter 312/313 successors. Incentive agreements are public records and are subject to council/board approval. Drafts are not commitments.

Site selection RFI responses often request confidential treatment. Verify confidentiality posture before sharing publicly.
