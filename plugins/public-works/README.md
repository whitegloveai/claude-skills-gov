# public-works

Claude skills for public works. Built for public works directors, engineers, fleet and facility managers, water utility staff, and capital improvement program leads.

## Skills in this plugin

| Skill | Purpose |
|---|---|
| [work-order-summarizer](skills/work-order-summarizer) | Summarizes a batch of work orders for a weekly or monthly operations report |
| [maintenance-log-analyzer](skills/maintenance-log-analyzer) | Reviews maintenance logs for trends, recurring failures, and asset condition signals |
| [capital-improvement-plan-drafter](skills/capital-improvement-plan-drafter) | Drafts the project narrative for a CIP entry: scope, justification, cost, schedule |
| [service-request-classifier](skills/service-request-classifier) | Classifies a citizen service request and routes it to the correct work group |
| [infrastructure-condition-report](skills/infrastructure-condition-report) | Drafts an infrastructure condition report summarizing inspection data |

## Install

```
/plugin install public-works@whitegloveai-gov
```

## Compliance

These skills support routine operational and CIP documentation. They do not certify engineering work product. Anything requiring a P.E. seal must be signed by a licensed engineer.

## Notes on Examples

Examples in this plugin reference the fictional City of Cedar Ridge, TX for continuity across skills. The skill logic is jurisdiction-agnostic — adapt example specifics to your city.
