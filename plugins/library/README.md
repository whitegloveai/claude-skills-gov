# library

Claude skills for public library staff. Built for library directors, programming librarians, collection development staff, and outreach coordinators at municipal and county public libraries.

## Skills in this plugin

| Skill | Purpose |
|---|---|
| [program-description-writer](skills/program-description-writer) | Drafts a program description for the library calendar, social, or printed materials |
| [collection-development-justifier](skills/collection-development-justifier) | Drafts the rationale for a collection development decision (acquisition, weeding, or reconsideration response) |
| [community-partnership-proposal](skills/community-partnership-proposal) | Drafts a partnership proposal between the library and a school, nonprofit, or community organization |

## Install

```
/plugin install library@whitegloveai-gov
```

## Compliance and intellectual freedom

The collection-development-justifier skill is built around the American Library Association's Library Bill of Rights, the Freedom to Read Statement, and standard reconsideration procedures. Outputs are designed to support, not replace, a library's professional collection development policy and intellectual freedom posture.

Program descriptions intended for citizen-facing channels include the TRAIGA-required AI-interaction disclosure where the citizen interacts with AI directly. Static program descriptions do not require disclosure but should be reviewed by staff before publication.
