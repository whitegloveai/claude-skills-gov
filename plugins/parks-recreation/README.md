# parks-recreation

Claude skills for parks and recreation. Built for parks directors, recreation supervisors, athletics coordinators, and program staff at city parks departments.

## Skills in this plugin

| Skill | Purpose |
|---|---|
| [program-flyer-drafter](skills/program-flyer-drafter) | Drafts copy for a recreation program flyer: dates, age group, fees, registration, accessibility |
| [volunteer-recognition-writer](skills/volunteer-recognition-writer) | Drafts a volunteer recognition letter or social post for an individual or group |
| [facility-rental-response](skills/facility-rental-response) | Drafts a facility rental inquiry response with availability, rates, and policies |
| [sponsorship-proposal-helper](skills/sponsorship-proposal-helper) | Drafts a sponsorship proposal for a program, event, or facility naming opportunity |

## Install

```
/plugin install parks-recreation@whitegloveai-gov
```

## Compliance

Citizen-facing flyers and program materials produced through these skills include the TRAIGA-required AI-interaction disclosure where the citizen is interacting with AI directly (e.g., chatbot-style intake). Static flyers and proposals do not require disclosure but should be reviewed by a human before distribution.

Sponsorship and naming agreements typically require council or board approval and may trigger public bidding or transparency rules depending on jurisdiction.
