# hr

Claude skills for local government HR. Built for HR directors, generalists, benefits coordinators, and risk managers handling personnel matters in the public sector.

## Skills in this plugin

| Skill | Purpose |
|---|---|
| [job-description-writer](skills/job-description-writer) | Drafts a job description with essential functions, minimum qualifications, and FLSA classification |
| [policy-analyzer](skills/policy-analyzer) | Reviews an HR policy against current law and identifies gaps or risk |
| [onboarding-packet-builder](skills/onboarding-packet-builder) | Assembles a new-hire onboarding packet checklist |
| [employee-handbook-updater](skills/employee-handbook-updater) | Drafts updates to a handbook section based on legal or policy changes |
| [performance-eval-helper](skills/performance-eval-helper) | Drafts a performance evaluation narrative from raw notes and goal data |
| [ada-accommodation-drafter](skills/ada-accommodation-drafter) | Drafts an ADA reasonable accommodation interactive-process letter or accommodation memo |

## Install

```
/plugin install hr@whitegloveai-gov
```

## Compliance

These skills target FLSA, ADA/ADAAA, Title VII, FMLA, and Texas-specific public-sector employment law (including Texas Government Code Chapter 614 for first responders and Texas Local Government Code provisions for civil service). HR decisions require human judgment and often legal review. Drafts are starting points.

## Notes on Examples

Examples in this plugin reference the fictional City of Cedar Ridge, TX for continuity across skills. The skill logic is jurisdiction-agnostic — adapt example specifics to your city.
