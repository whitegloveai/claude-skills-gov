# legal

Claude skills for the city attorney's office. Built for city attorneys, assistant city attorneys, and contract compliance staff. Output here is paralegal-grade drafting; an attorney reviews everything before it leaves the office.

## Skills in this plugin

| Skill | Purpose |
|---|---|
| [contract-redline-summary](skills/contract-redline-summary) | Summarizes vendor redlines on a contract draft with risk flags |
| [ordinance-impact-analyzer](skills/ordinance-impact-analyzer) | Analyzes a proposed ordinance for conflicts with existing code, charter, and state law |
| [tpia-legal-review](skills/tpia-legal-review) | Reviews a TPIA response for required AG referral, exception assertions, and redaction completeness |
| [litigation-summary-helper](skills/litigation-summary-helper) | Drafts a privileged-work-product litigation summary for council executive session |
| [interlocal-agreement-drafter](skills/interlocal-agreement-drafter) | Drafts an interlocal cooperation agreement under Tex. Gov't Code Ch. 791 |

## Install

```
/plugin install legal@whitegloveai-gov
```

## Compliance and disclaimer

Output from these skills is **not legal advice** and is not a substitute for the judgment of a licensed attorney. The skills are drafting aids for use by attorneys and authorized legal staff. Anything generated should be reviewed by a licensed attorney before being relied upon.

## Notes on Examples

Examples in this plugin reference the fictional City of Cedar Ridge, TX for continuity across skills. The skill logic is jurisdiction-agnostic — adapt example specifics to your city.
