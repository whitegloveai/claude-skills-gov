# Zoning Narrative Writer

> Drafts the substantive narrative behind a zoning request so the case file, P&Z packet, and council packet all carry the same defensible reasoning. Built for planners who have done the analysis and need to put it on paper.

## When to Use

- "Draft the narrative for the 1100 Main Street rezoning case."
- "Write up the comp plan conformance section for the special-use permit at 215 Elm."
- "I need the surrounding land use and compatibility analysis for this rezoning case."
- "Put together the staff narrative for the PD amendment going to P&Z next month."
- A rezoning, special-use permit, or planned development case is moving toward a noticed public hearing and the case file needs the written justification that will become an exhibit if the decision is appealed under Tex. Loc. Gov't Code § 211.011.

## Inputs Needed

Required:

- Case number and case type (rezone, SUP, PD, PD amendment)
- Subject property: address, legal description, acreage, current zoning, proposed zoning
- Applicant name and applicant's stated reason for the request
- Comprehensive plan future land use designation for the property
- Surrounding land uses and zoning on all four sides (or all bounding directions)
- Infrastructure context: water and sewer availability, street classification, traffic counts if known, school district capacity if residential
- Public input received to date (calls, emails, neighborhood meetings, written comments)
- The specific criteria the city's zoning ordinance directs staff to address

Optional:

- Prior zoning history on the parcel
- Pending or recent cases nearby
- Concept plan, elevations, or site plan if submitted

If the user has not provided the comp plan designation, the surrounding context, or the ordinance criteria, ask before drafting. A narrative without the criteria is not a narrative.

## Process

1. Confirm the case type and identify the controlling ordinance section. Rezoning and SUP cases proceed under Tex. Loc. Gov't Code Ch. 211; the local zoning ordinance supplies the substantive criteria.
2. Draft the property and request summary in three to five sentences. Use the legal description, not just the street address.
3. Write the comprehensive plan conformance section. Quote the future land use designation, name the plan and adoption year, and explain how the request advances or departs from the designation.
4. Map the surrounding land uses. Walk the four cardinal directions; describe both the zoning and the actual ground use on each side.
5. Write the compatibility analysis. Address height, scale, intensity, hours of operation, traffic generation, and any nuisance factors. Be specific about transitions and buffers.
6. Write the public infrastructure section. Address water, sanitary sewer, storm drainage, transportation, and schools where relevant. If a traffic impact analysis exists, summarize the finding; if not, note the trip generation expected from the proposed use.
7. Summarize public input. Identify the number and substance of comments, the dates of any neighborhood meetings, and whether opposition has reached the § 211.006 protest threshold of 20% of nearby owners.
8. Draft findings keyed one-to-one to the ordinance criteria. Each finding states the criterion, the supporting facts, and the conclusion. Bare conclusions are not findings.
9. Close with the staff recommendation: approval, approval with conditions, or denial. Tie the recommendation back to the findings.

## Output Format

Plain markdown. No emoji. No bolding except section headers. Sections in this order:

```
CASE NUMBER: [Z-YYYY-NN]
APPLICANT: [Name]
PROPERTY: [Address; legal description; acreage]
REQUEST: [From X zoning to Y zoning]

1. PROPERTY AND REQUEST SUMMARY
2. COMPREHENSIVE PLAN CONFORMANCE
3. SURROUNDING LAND USES
4. COMPATIBILITY ANALYSIS
5. PUBLIC INFRASTRUCTURE IMPACT
   5a. Water and sanitary sewer
   5b. Transportation
   5c. Stormwater
   5d. Schools (if residential)
6. PUBLIC INPUT SUMMARY
7. FINDINGS
   7a. [First ordinance criterion] — Finding
   7b. [Second ordinance criterion] — Finding
   (continue for each criterion)
8. RECOMMENDATION

PREPARED BY: [Planner name and title], [Date]
REVIEWED BY: [Planning Director], [Date]
```

Length: typically 4–8 pages for a rezoning, 2–4 pages for a routine SUP, longer for a planned development. Findings should be written so a reasonable person reading only the findings can understand why staff reached the recommendation.

## TRAIGA & Compliance Notes

**AI use disclosure — transparency best practice (not a § 552.051 trigger):** Tex. Bus. & Comm. Code § 552.051(b)–(d) requires disclosure when a citizen *interacts directly with an AI system* (e.g., a chatbot, automated voice assistant, or AI-driven service interface). The disclosure trigger is *interaction*, not *output*. This skill produces a static document drafted with AI assistance; it does not put a citizen in direct interaction with AI, so § 552.051 is not engaged.

However, transparency about AI assistance in document drafting is sound practice and should align with the city's adopted AI use policy and the GovAI Coalition's guidance on documenting AI tool use. Consider a brief notation on documents distributed publicly or to the governing body: *"AI-assisted draft, reviewed by [staff title and name]."* This is a policy choice, not a statutory mandate.

**Quasi-judicial exhibit risk:** Zoning decisions are reviewable by writ of certiorari under Tex. Loc. Gov't Code § 211.011, with a 10-day filing deadline. The staff narrative, findings, and any maps or exhibits attached can become part of the certified record. Cite the comp plan and ordinance accurately. Do not editorialize.

**Fair Housing Act:** Zoning decisions that touch group housing, supportive housing, or any protected class issue are reviewable under 42 U.S.C. § 3601 et seq. When the proposed use is residential and serves a protected population, the narrative must avoid any language that could read as a proxy for discriminatory intent. Route those cases through the city attorney before the narrative is finalized.

**Protest petition:** Tex. Loc. Gov't Code § 211.006(d) requires a three-fourths council supermajority to approve a rezoning when at least 20% of the owners of land within 200 feet protest in writing. The narrative should note whether a protest petition is pending and the threshold count if so.

**Retention:** Zoning case files are retained permanently under Texas State Library Local Government Retention Schedule DC, Item 1100-04 (Zoning Case Files).
