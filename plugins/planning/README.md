# planning

Claude skills for planning and development services. Built for planners, planning directors, P&Z board liaisons, and development services staff handling zoning, permits, and land-use review.

## Skills in this plugin

| Skill | Purpose |
|---|---|
| [zoning-narrative-writer](skills/zoning-narrative-writer) | Drafts the narrative for a zoning change or special-use permit application |
| [permit-application-reviewer](skills/permit-application-reviewer) | Reviews a permit application for completeness against checklist and code |
| [site-plan-summary](skills/site-plan-summary) | Summarizes a site plan submittal for staff or board review |
| [public-hearing-notice](skills/public-hearing-notice) | Drafts a public hearing notice for a planning matter, including statutory posting requirements |
| [variance-request-analyzer](skills/variance-request-analyzer) | Analyzes a variance request against the unique-hardship standard |
| [staff-report-writer](skills/staff-report-writer) | Drafts a planning staff report for P&Z or council with findings and recommendation |

## Install

```
/plugin install planning@whitegloveai-gov
```

## Compliance

These skills target Texas Local Government Code Chapters 211 (zoning) and 212 (subdivision), TOMA, and the procedural requirements that apply to land-use decisions. Findings of fact and conclusions of law in any document going to P&Z or council should be reviewed by the city attorney.
