# Procurement RFP Builder

> Drafts the scope of work, evaluation criteria, and submission requirements for an RFP. Built for purchasing staff and department heads who must launch a competitive procurement that satisfies Tex. Loc. Gov't Code Ch. 252.

## When to Use

- "Build the RFP for the FY2026 street overlay program — approximately $1.8M, twelve lane-miles."
- "Draft an RFP for janitorial services for City Hall and three satellite buildings."
- "Write the scope and evaluation criteria for the citizen survey vendor selection."
- "Convert this scope memo into a full RFP package — we want to advertise by month-end."
- "Create the RFP for a financial software replacement under the high-tech exception."
- A purchasing manager is preparing a procurement that exceeds $50,000 (current LGC Ch. 252 threshold) and needs the package built to the city's standard format.

## Inputs Needed

Required:

- Entity name and procurement title
- Estimated budget range (or budgeted amount)
- Scope summary: what is being procured, where it will be performed, and the period of performance
- Functional requirements: must-haves vs nice-to-haves
- Schedule: advertisement, pre-proposal, proposals due, evaluation, award, contract execution, start date
- Evaluation factors and recommended weights

Optional:

- Funding source (general fund, bond proceeds, grant) — federal funds trigger 2 C.F.R. § 200.317–.327 layered on top
- Whether prevailing wage applies (Tex. Gov't Code Ch. 2258 for most public works construction)
- Whether HUB participation goals apply (state-funded projects under Tex. Gov't Code Ch. 2161)
- Set-aside or preference programs
- Insurance and bonding requirements (especially Tex. Gov't Code Ch. 2253 payment and performance bonds for construction over $100,000)

If the user has not provided the procurement method, the budget range, or the schedule, ask before drafting.

## Process

1. Confirm the procurement method. Tex. Loc. Gov't Code Ch. 252 permits competitive sealed bids for most construction and goods, competitive sealed proposals for services where price is not the only factor, and alternative delivery methods under Tex. Gov't Code Ch. 2269. The method drives the document format.
2. Confirm thresholds. LGC § 252.021 sets a $50,000 fiscal-year threshold for goods or general services. Procurements over the threshold require advertisement once a week for two consecutive weeks before bid opening.
3. Draft the Project Overview: purpose, background, expected outcome, funding source.
4. Draft the Scope of Work: deliverables, performance standards, period of performance, location, milestones. This is the most-edited section.
5. Draft Submittal Requirements: cover letter, technical approach, qualifications, references, price proposal, required forms. Page limits help reviewers.
6. Draft Evaluation Criteria with weights that sum to 100. For competitive sealed proposals, price must be a factor but is rarely 100 percent.
7. Draft Schedule and Submission Instructions: dates, time zone, submission method, late-submission policy, public opening rules.
8. Draft Required Forms: Form CIQ (Tex. Loc. Gov't Code Ch. 176), HB 89 Israel-divestment certification (Tex. Gov't Code Ch. 2271), Ch. 2252 Subch. F certification, debarment (federal funds), W-9, franchise tax good standing.
9. Draft Contract Terms Summary and program-specific overlays (prevailing wage, bonds, federal flow-downs if applicable).

## Output Format

Plain markdown. Tone: formal, precise, complete. The reader is a vendor pricing the work. Ambiguity costs money — in protests, in change orders, and in unqualified bids.

Length: 6 to 15 pages depending on complexity. Structure:

```
[ENTITY NAME]
REQUEST FOR PROPOSALS
RFP NO. [number]
[PROCUREMENT TITLE]

ADVERTISEMENT DATES: [first] / [second]
PROPOSAL DUE DATE: [Date], [Time] [Time zone]

1. PROJECT OVERVIEW
   1.1 Purpose
   1.2 Background
   1.3 Funding Source

2. SCOPE OF WORK
   2.1 Deliverables
   2.2 Performance Standards
   2.3 Period of Performance
   2.4 Location
   2.5 Milestones

3. SUBMITTAL REQUIREMENTS

   3.6 Required Forms

4. EVALUATION CRITERIA
   | Criterion | Weight |

   | Milestone | Date |

   [Term length, renewal options, insurance, bonds, termination,

   [List with statutory citations.]

   [Method, format, late-submission policy, public opening rules.]
```

## TRAIGA & Compliance Notes

**AI use disclosure — transparency best practice (not a § 552.051 trigger):** Tex. Bus. & Comm. Code § 552.051(b)–(d) requires disclosure when a citizen *interacts directly with an AI system* (e.g., a chatbot, automated voice assistant, or AI-driven service interface). The disclosure trigger is *interaction*, not *output*. This skill produces a static document drafted with AI assistance; it does not put a citizen in direct interaction with AI, so § 552.051 is not engaged.

However, transparency about AI assistance in document drafting is sound practice and should align with the city's adopted AI use policy and the GovAI Coalition's guidance on documenting AI tool use. Consider a brief notation on documents distributed publicly or to the governing body: *"AI-assisted draft, reviewed by [staff title and name]."* This is a policy choice, not a statutory mandate.

**Texas procurement law:**

- Tex. Loc. Gov't Code Ch. 252 — competitive procurement requirements for cities, including the $50,000 threshold and the once-a-week-for-two-consecutive-weeks advertisement rule. § 252.044 requires public bid opening.
- Tex. Gov't Code Ch. 2269 — alternative project delivery methods for public works (CMAR, design-build, JOC, CSP).
- Tex. Gov't Code Ch. 2258 — prevailing wage rates for public works.
- Tex. Gov't Code Ch. 2253 — payment and performance bonds for public works over $100,000.
- Tex. Loc. Gov't Code Ch. 176 — Form CIQ vendor conflict-of-interest filing; companion Form CIS for officers.
- Tex. Gov't Code Ch. 2271 — Israel anti-divestment certification for contracts of $100,000+ with companies of 10+ employees.
- Tex. Gov't Code Ch. 2252 Subch. F — certification regarding business with Iran, Sudan, and foreign terrorist organizations.

**Federal funds overlay:** When federal funds are involved, layer 2 C.F.R. §§ 200.317–.327: federal procurement standards, Build America Buy America, Davis-Bacon, Section 3, and program-specific certifications. State procedures govern only if more restrictive than the federal floor.

**Public records:** RFPs and submitted proposals are public records under Tex. Gov't Code Ch. 552. Trade secret material may be protected under § 552.110 if the vendor properly designates it.

**Retention:** Procurement files retain 7 years after closeout (Texas State Library Local Government Retention Schedule GR, Item 1075). Federal-funded procurements retain 3 years from final closeout under 2 C.F.R. § 200.334.

**Verify before advertisement:** Procurement method matches the underlying authority; advertisement dates satisfy the two-week rule; required forms listed are current versions; evaluation weights sum to 100; conflict-of-interest questionnaire requirement is included.
