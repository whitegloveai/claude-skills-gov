# finance

Claude skills for municipal finance. Built for finance directors, budget analysts, grant managers, procurement officers, and CFOs in cities, counties, and special districts.

## Skills in this plugin

| Skill | Purpose |
|---|---|
| [budget-narrative-writer](skills/budget-narrative-writer) | Drafts the narrative section of a department budget from goals, KPIs, and prior-year actuals |
| [variance-analyzer](skills/variance-analyzer) | Explains material variances in a monthly or quarterly financial report |
| [grant-application-drafter](skills/grant-application-drafter) | Drafts a federal or state grant application narrative against a published NOFO |
| [procurement-rfp-builder](skills/procurement-rfp-builder) | Drafts an RFP scope, evaluation criteria, and submission requirements |
| [transparency-report-generator](skills/transparency-report-generator) | Drafts a Texas Comptroller transparency-stars-style summary or comparable disclosure |
| [audit-response-helper](skills/audit-response-helper) | Drafts management responses to audit findings |
| [financial-disclosure-formatter](skills/financial-disclosure-formatter) | Formats required financial disclosure statements for public officials |

## Install

```
/plugin install finance@whitegloveai-gov
```

## Compliance

These skills assume GASB-compliant accounting, Texas-specific procurement law (Local Gov. Code Ch. 252), and federal grant rules under 2 CFR Part 200 (Uniform Guidance) where applicable. Outputs are drafts for finance staff and external auditor review.
