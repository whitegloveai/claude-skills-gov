# Example: Council Packet Cover and Index — Cedar Ridge June 9, 2026

This is a worked example for the council-packet-prep skill. It supports the same fictional Cedar Ridge regular meeting as the agenda-builder example.

## Input

- The June 9, 2026 agenda (see agenda-builder example)
- Item types and packet support known to date
- City Manager Daniel Ahn signs the cover memo
- The packet is being finalized on Friday, June 5, 2026

## Notes for the User

- Confirm with the City Attorney before posting the public version of the packet that no privileged content from items 7a or 7b leaked into the public-facing files.
- The "separate packet" line is the standard way to indicate executive session materials without disclosing their contents on the public-facing index.
- If a packet contains an item that ultimately gets pulled before the meeting, append a "Pulled — see [date] supplemental" note to the History rather than removing the entry.
