# Standard Packet Attachments by Item Type

This is a working reference for the council-packet-prep skill. It maps common agenda item types to the attachments that should be in the packet, with the statutory or charter driver where applicable.

This list is the baseline. The city manager and city attorney may require more depending on local rules and the complexity of the item.

## Contract / Procurement

| Required | Recommended |
|---|---|
| Cover memo (item summary, fiscal impact, term, scope) | Vendor responses or pricing comparison |
| Draft contract (final form proposed for execution) | Insurance certificates |
| Bid tabulation or selection memo (for competitive procurements under Tex. Loc. Gov't Code Ch. 252 or Ch. 271) | Vendor performance history |
| City Attorney "Approved as to form" sign-off | |

Statutory drivers: Tex. Loc. Gov't Code Ch. 252 (general competitive bidding); Ch. 271 (purchasing methods for buildings); Government Code Ch. 2269 (construction); Ch. 791 (interlocal cooperation).

## Zoning Amendment / Special Use Permit / Variance

| Required | Recommended |
|---|---|
| Staff report (factual findings, applicable standards, recommendation) | Site photographs |
| Plat or site plan | Traffic study (where applicable) |
| Notice of public hearing (mailed and published) with proof of publication | Surrounding zoning map |
| List of property owners notified (200 ft) | Drainage / utility analysis |
| P&Z Commission recommendation | |
| Written protests received (count toward § 211.006(d) threshold) | |
| Applicant submission | |

Statutory drivers: Tex. Loc. Gov't Code Ch. 211, especially § 211.006 (notice) and § 211.007 (200 ft notice).

## Budget Adoption or Amendment

| Required | Recommended |
|---|---|
| Proposed budget document or amendment summary | Comparative variance analysis |
| Notice of public hearing with proof of publication | Department-level narrative |
| Tax rate calculation (if budget adoption) | Five-year forecast |
| Fiscal note describing the change in fund balance | |

Statutory drivers: Tex. Loc. Gov't Code Ch. 102 (municipal budget); Tex. Tax Code Ch. 26 (tax rate).

## Items That Should NOT Be in the Public Packet

- Attorney-client privileged memos prepared for executive session deliberation
- Personnel files for items under § 551.074
- Real estate negotiation memos for items under § 551.072
- Information whose disclosure would compromise security under § 551.076 or § 551.089
- Personal information that must be redacted under Tex. Gov't Code § 552.117 (peace officer addresses), § 552.130 (driver's license numbers), § 552.147 (Social Security numbers), and similar provisions

These materials go in a separate executive session packet routed only to council, the city manager, and counsel.

## Practical Completeness Checklist

For each item, before packet release, confirm:

1. Cover memo or staff report is present.
2. Required attachments per the table above are present.
3. Statutory notice (where required) is in the packet with proof of publication.
4. City Attorney sign-off is documented (where required).
5. Page numbers in the index match the actual packet.
6. Executive session items have been routed in a separate executive session packet.

This reference is informational and is not legal advice. Consult the city attorney for any item where the application of a statute is unclear.
